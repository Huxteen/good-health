How To Use
1. download the file(Zip, Github etc). 
2. Extract if its in zip format.
3. Save it to your root directory.
4. Open your shell command
5. Navigate to the folder directory where you have the manage.py file
6. Run this command "python manage.py runserver" remove the quote.
7. The on your preferred browser go to the address "http://127.0.0.1:8000/" remove the quote.
8. Viola! the app is running.
9. Create an account.
10. Enjoy your stay.
